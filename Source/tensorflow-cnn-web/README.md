# Tensorflow - Image Classification


see [TF Model reference](https://www.tensorflow.org/tutorials/image_retraining) for the inception model details



### Requirements
* Python >=2.7 or >=3.4
  * Tensorflow >=1.0 


### Usage
    $ pip install -r requirements.txt
    $ export FLASK_APP=lable_image.py
    $ python -m flask run
